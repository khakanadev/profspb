# ПрофСПб (profspb)

Одностраничный сайт профориентации для выпускников 11 класса **Санкт‑Петербурга**: введение, актуальность, короткий опрос с ориентиром по направлению и вузу, топ‑10 петербургских вузов с ссылками и FAQ.

**Стек:** React 19, TypeScript, Vite, styled-components.

## Запуск

```bash
npm install
npm run dev
```

Сборка и превью:

```bash
npm run build
npm run preview
```

Линт:

```bash
npm run lint
```

## Важно

Тексты и «ориентиры» по баллам **не заменяют** официальные сайты вузов и приёмную кампанию. Фотографии зданий взяты с Wikimedia Commons (см. подпись в разделе вузов).
